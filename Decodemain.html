How Does It Work?

Participants will be provided with a set of code snippets written in various programming languages, such as Python, Java, C++, and more. Each code snippet will contain one or more errors, ranging from syntax mistakes to logical flaws. Your mission is to debug the code by identifying and rectifying these errors within a specified time limit.

Points will be awarded for each successfully debugged code snippet. The participant with the highest score at the end of the event will be crowned the ERROR DEBUGGING Champion!

Why Participate?

  1. Hone Your Debugging Skills: Test your ability to identify and fix errors in code, an essential skill for any programmer.
  2. Expand Your Programming Knowledge: Encounter different programming languages and coding techniques, broadening your expertise.
  3. Competitive Environment: Compete against fellow programmers, challenging yourself to solve errors quickly and accurately.
  4. Prizes and Recognition: Stand a chance to win exciting prizes and gain recognition for your debugging prowess.
